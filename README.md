# PRAAgent
Performance Risk Assessment Scoring Tool

Install Streamlit package from python  
Open the terminal in VSCode  
Give the path where the streamlit is installed C:\Users\<user>\AppData\Roaming\Python\Python313\Scripts/   
To run the code use the command as "C:\Users\<user>\AppData\Roaming\Python\Python313\Scripts/streamlit run .\main.py"  
The App should open in a chrome browser.
